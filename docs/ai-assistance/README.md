# AI assistance artifacts

Tooling and prompt files used during development with AI coding assistants,
kept here for transparency. They are not part of the application.

- `frontend-claude-instructions.md` — working instructions given to Claude Code
  for the frontend (design tokens, stack conventions, route map).
- `stitch-prompt.md` — the prompt used to generate the initial UI mockups with
  Google Stitch. The resulting mockups live in `docs/design/stitch/`.

All AI-assisted code was reviewed and verified manually; the development
process is documented in `docs/dev-diary.md` and the per-script audits in
`audit/`.
